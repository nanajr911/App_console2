# Student Results Console Application

A simple C# console application that collects student details, calculates their total marks from **Class Test**, **Midsemester Exam**, and **Final Exam**, and then assigns a grade based on the total score.

## 📖 Features
- Prompt user to enter student details (name, department, program, course).
- Accept marks for class test, midsemester, and final exam.
- Calculate total marks.
- Assign grade based on total marks:
  - A: 80 and above
  - B: 70–79
  - C: 60–69
  - D: 51–59
  - F: 50 and below
- Display a formatted student result summary.


## 🛠️ Technologies Used
- **Language:** C#
- **Framework:** .NET Console Application
